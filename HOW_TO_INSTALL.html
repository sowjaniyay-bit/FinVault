📱 Install on Android or iPhone

FinVault can be installed directly from your browser — no Play Store or App Store needed. It works just like a real app with its own icon on your home screen.

1
Start the backend on your computer
Double-click MAKE_APP.bat (Windows) or run MAKE_APP.sh (Mac/Linux) in the FinVault folder.
2
Find your computer's IP address
On Windows: Press Win+R → type cmd → type ipconfig → look for "IPv4 Address" like 192.168.1.5
On Mac: System Settings → Wi-Fi → Details → IP Address
3
Open in phone browser (must be on same Wi-Fi!)
Android: Open Chrome → type: 192.168.1.5:5500/frontend/home_app.html
iPhone: Open Safari → type the same address
4
Add to Home Screen
The phone will ask to "Add to Home Screen" automatically. If not:
🤖 Android Chrome: Tap ⋮ menu → "Add to Home Screen" → "Install"
🍎 iPhone Safari: Tap Share button (box with arrow) → "Add to Home Screen" → "Add"

🪟 Windows

Windows Desktop App (.exe)

Turn FinVault into a proper Windows application that installs like any other program.

1
Install Node.js from nodejs.org
2
Run: npm install electron electron-builder -D
3
Run: npm run build
4
Get FinVault-Setup.exe in the dist folder
⭐⭐ Medium~30 min
📱 Android APK

Android App (APK file)

Build a real Android APK you can install on any Android phone or submit to the Play Store.

1
Install Node.js + Android Studio
2
Run: npm install @capacitor/cli
3
Run: npx cap add android
4
Open Android Studio → Build → Generate APK
⭐⭐⭐⭐ Hard~2 hours

❓ Common Questions

My phone can't connect to the app ▼
Make sure your phone and computer are on the same Wi-Fi network. Also check Windows Firewall — open Control Panel → Windows Defender Firewall → Allow an app → make sure Python is allowed. Try turning off firewall temporarily to test.
What is the backend / why do I need Python? ▼
FinVault has two parts: the frontend (the screens you see — HTML/CSS) and the backend (the brain that saves your data — Python Flask). The backend stores transactions in a database. Without it, the app shows the UI but can't save or load any data.
Can I use it without running Python every time? ▼
Yes! Use the Python Desktop App method (PyInstaller + pywebview). It bundles the Python backend and the frontend into one .exe file. Double-click it and everything starts automatically — no terminal needed.
Will my data be safe? Where is it stored? ▼
Your data is stored locally in backend/finance.db — a SQLite file on your own computer. Nothing is sent to the internet unless you deploy to the cloud yourself. Your financial data never leaves your machine.
I see "Add to Home Screen" — is it really an app? ▼
Yes! When you add it to your home screen from Chrome or Safari, it opens in its own window without the browser address bar — just like a real app. It has its own icon, loads instantly, and works in full screen. This is called a Progressive Web App (PWA).
How do I back up my data? ▼
Simply copy the file backend/finance.db to a safe location (USB drive, Google Drive, etc.). To restore, just put it back in the backend folder. You can also export all transactions as CSV from the Transactions page anytime.
