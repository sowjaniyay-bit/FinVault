How to Run FinVault

FinVault needs two things running at the same time: the backend server (Python) and the frontend (your browser). Follow these steps exactly.

1
Open a Terminal in the project folder
In VS Code → Terminal → New Terminal. Make sure you're in the FinVault_FIXED folder.
2
Install dependencies (first time only)
cd backend
pip install flask bcrypt PyJWT
3
Start the backend server
python app.py
You should see: Running on http://127.0.0.1:5000 — keep this terminal open!
4
Open the frontend using VS Code Live Server
Right-click frontend/login.html → Open with Live Server.
⚠️ Always open via Live Server (port 5500), not by double-clicking the file.
⚠️ Don't open HTML files by double-clicking — that uses file:// protocol which blocks the backend connection. Always use Live Server or a local web server.
→ Go to Login Page
